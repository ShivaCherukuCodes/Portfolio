Aadhaar Vault Solution (Vsoft Technologies)

Description:

Aadhaar Vault is a specialized solution designed to securely store Aadhaar numbers for Aadhaar-related applications. It functions as an 'Aadhaar data vault' employing stringent security measures. Each Aadhaar number is associated with a unique reference key, serving as an additional layer of security. This reference key is utilized across applications as a substitute for Aadhaar numbers within their respective data storage systems, ensuring confidentiality and integrity of sensitive information.

Responsibilities:

  • • Managed key creation, updates, and deletions employing both symmetric and asymmetric encryption/decryption in Hardware Security Modules (HSM) to fortify data security.
